To run:

1. Install XAMPP
2. Run XAMPP Control Panel, start Apache and MySQL modules
3. Place unzipped code folder in htdocs of XAMPP, should look something like this 
	-> C:\XAMPP\htdocs\CSCI4560_GradeBookProject
4. Navigate to http://localhost/phpmyadmin in browser
5. Import SQL file to create database
6. Navigate to http://localhost/CSCI4560_GradeBookProject/
7. Use one of the students or teacher logins, stored in plain text in the 'users' table of the database to view functionalties
8. You may also register a new user, but this user won't have courses/grades associated with it

*Assuming you haven't changed the folder name from zip file of code
